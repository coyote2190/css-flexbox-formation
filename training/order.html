Order

Elle permet de réordonner au choix chacun des éléments (flex-item)

La propriété justify-content a pour valeur un nombre entier:
order: "un nombre entier"

Example 1 :

Code

.item01 { order: 0; }
.item02 { order: 0; }
.item03 { order: 0; }
.item04 { order: 0; }
.item05 { order: 0; }

N°1 - order : 0
N°2 - order : 0
N°3 - order : 0
N°4 - order : 0
N°5 - order : 0
