Flex-basis

Elle définit les dimensions (hauteur/largeur) d’un élément (flex-item)

La valeur par défaut est auto.

Exemple 1 :

Code

.item01 { flex-basis: auto; flex-grow: 1; flex-shrink: 1; }
.item02 { flex-basis: auto; flex-grow: 1; flex-shrink: 1; }
.item03 { flex-basis: auto; flex-grow: 1; flex-shrink: 1; }
